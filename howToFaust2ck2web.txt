earlier notes on how to do this step-by-step
  from wherever it lives in the universe, get started with a copy of 
  your src .dsp in a scratch directory
  for example starting from freeverbStereo.dsp in this repo
  the .dsp name at this point can be capitalized or not
cp freeverbStereo.dsp /tmp/

  edit your src and make sure chugin name is capitalized
gedit /tmp/freeverbStereo.dsp
  [ declare name "FreeverbStereo"; ]

  generate a template, matching the capitalized name, and cd into it
chuginate/chuginate FreeverbStereo
cd FreeverbStereo

  copy your edited src and rename it to FreeverbStereoF
cp /tmp/freeverbStereo.dsp FreeverbStereoF.dsp

  compile the edited and renamed src in the working directory 
faust2ck FreeverbStereoF.dsp

  copy the new faust-generated .cpp file over the old chuginate template .cpp file
cp FreeverbStereoF.dsp.cpp FreeverbStereo.cpp

  if this is not the first make run, remove the .wasm so make will trigger again
rm FreeverbStereo.chug.wasm
  
  compile the .wasm
make web

  copy the .wasm to where you'll be testing it with webchuck
cp -p FreeverbStereo.chug.wasm ~/webchuck-github/test/testFiles/
